本站建站者在对来自中国共产党新闻网（实际来源于宣讲家网）马克思恩格斯全集、列宁全集、马主义文库的部分排版后的网页、马克思恩格斯文集电子书、贺麟全集电子书、杨一之译黑格尔《逻辑学》电子书、小逻辑英译本电子书等处的文本重新排版后，为高效利用个人资源，将建站者本人所制作的HTML及HTML的压缩包（EPUB）上传至本站，以便下载、在线使用有关格式的文件。由于建站者个人水平有限，本站在校订、排版、代码等方面难免有各种不足，还请各位网友批评指正。
