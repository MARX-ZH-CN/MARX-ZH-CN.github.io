本站建站者在对来自中国共产党新闻网（实际来源于宣讲家网）马克思恩格斯全集、列宁全集、马主义文库的部分排版后的网页、马克思恩格斯文集电子书、贺麟全集电子书、杨一之译黑格尔《逻辑学》电子书、小逻辑英译本电子书等处的文本重新排版后，为高效利用个人资源，将所制作的HTML及HTML的压缩包（EPUB）上传至本站，以便建站者本人下载、在线使用有关格式的文件。由于个人能力有限，本站网页在内容、代码上有各种疏漏、错误，也请各位访问本站的网友注意，并欢迎提出更正、编辑、改进的建议。

1.列宁全集8、13-28卷尚未排版完毕——粗体、着重号部分尚未标记，44-60卷内容、排版格式尚不齐全。马恩全集46-49卷、马恩文集6-8卷也仅录入文本和部分格式，尚缺排版；
2.全站内容均尚未详细校对，可能有文字输入或排版上的差错；
3.由于html静态页代码的技术限制，无法准确还原原文内容的各种格式；
4.为节省空间与保持版面清爽，以及技术限制，未记录各文章、内容原书对应的页码。
因此，本站各网页并不能完全取代、视同原书有关篇目的内容。如各位网友在有引用、摘抄、标注等学习需要，请务必翻阅原书，仔细对照原书内容、排版、页码，以原书为准，切不可完全不顾原书。马恩著作可前往德古意特出版社、马克思恩格斯全集历史考证版（MEGA）编辑委员会、Zeno.org马克思、恩格斯著作页、MEGAdigital等处参考相关内容的德语原文。
