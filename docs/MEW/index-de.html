说明

　　MARX ENGELS WERKE（MEW，直译为马克思恩格斯著作）目前共出版44卷。其中前43卷由民主德国狄茨出版社出版，1-39卷篇目按照马恩全集俄文第二版编排，与马恩全集中文第一版篇目、卷次划分一致，40-43卷为增补卷，包含部分马克思恩格斯早期著作、政治经济学手稿，编排顺序与中文版不同，44卷及之后各卷由德国罗莎·卢森堡基金会续编。同时卢森堡基金会也在组织重订、再版1-43各卷。
　　而以上网页的内容均来源于东德版MEW，由mlwerke.de自1998年起整理录入，非罗莎卢森堡基金会重订版，缺26-39、41、43卷内容，以及数字方括号注释，且录入年代较为久远，格式较不规整，部分文本经多次转载（爬取）后产生乱码。
　　综上原因，以上网页内容仅供参考。柏林卡尔·狄茨出版社计划于2026年推出MEW1-44卷的电子版，如有需要，届时请以此官方版为准。
